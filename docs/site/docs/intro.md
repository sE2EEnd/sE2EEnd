# Introduction

**sE2EEnd** is an open-source, end-to-end encrypted file transfer solution. The server never sees encryption keys or plaintext data — encryption happens entirely in the browser using the native Web Crypto API (AES-256-GCM).

## Key features

- **Zero-knowledge architecture** — the server stores only ciphertext
- **Enterprise authentication** via Keycloak (OAuth2 / OIDC)
- **Password-protected transfers**
- **Per-transfer download limits and expiration dates**
- **Instant revocation**
- **Admin dashboard**
- **Multi-language UI** (EN, FR)
